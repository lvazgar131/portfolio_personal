Mi experiencia

Desarrollador web (backend)

Diseño y desarrollo de aplicaciones backend robustas y escalables utilizando Python, Django y Flask. Implementación de APIs REST, autenticación, lógica de negocio y conexión con bases de datos relacionales y no relacionales, siguiendo buenas prácticas y arquitectura limpia.

Python Django Flask REST APIs
DevOps y contenedores

Experiencia en la creación y gestión de entornos de desarrollo y producción mediante Docker, facilitando despliegues consistentes y escalables. Automatización de procesos, manejo de configuraciones y optimización del rendimiento de aplicaciones backend.

Docker Docker compose Linux CI/CD
Cloud y AWS

Despliegue y administración de aplicaciones backend en la nube usando AWS. Configuración de servicios como EC2, S3, RDS y IAM, asegurando disponibilidad, seguridad y escalabilidad para aplicaciones modernas.

AWS EC2 S3 Cloud
